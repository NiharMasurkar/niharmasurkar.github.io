PROFILE PHOTO
-------------
Add your About-section portrait here as:

  profile.jpg   (or .png)

Then make sure _config.yml has:

  about:
    photo: /assets/media/profile.jpg

A cut-out style photo (transparent or plain background, head-and-shoulders)
looks best sitting on the colored disc, like the reference. Until you add it,
the site shows a "your photo goes here" placeholder disc.

PROJECT MEDIA
-------------
Drop project photos and videos here, then reference them in _data/projects.yml:

  media: /assets/media/your-file.jpg
  media_type: image

or for video:

  media: /assets/media/your-clip.mp4
  media_type: video
  poster: /assets/media/your-thumb.jpg   (optional)
